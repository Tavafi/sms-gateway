## Why

* Increase development productivity.
* Log independent of application layer.
* Failover.

## Adapters

| Name        | Key           | Description                                     |
| ----------- | ------------- | ----------------------------------------------- |

## Features

* Send local for testing during application development.
* One interface rule them all.
* Administrator Web UI
